Dados Pessoais

O nome é obrigatório. O nome deve ter pelo menos 3 caracteres.
O CPF é obrigatório. Informe um CPF com 11 dígitos numéricos.
O telefone é obrigatório. Informe um telefone válido.
O e-mail é obrigatório. Informe um e-mail válido.

Dados da Academia

O nome da academia é obrigatório.
O CNPJ é obrigatório. Informe um CNPJ com 14 dígitos numéricos.
O endereço é obrigatório.
O bairro é obrigatório.
A cidade é obrigatória.
A UF é obrigatória.
O plano é obrigatório.

Dados de Acesso

O login é obrigatório. O login deve ter pelo menos 3 caracteres.
A senha é obrigatória. A senha deve ter pelo menos 6 caracteres.
